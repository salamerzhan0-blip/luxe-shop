# LUXE SHOP — E-Commerce Platform

Полноценная платформа электронной коммерции с бэкендом на Go, фронтендом на HTML/CSS/JS и базой данных SQLite.

---

## 📁 Структура проекта

```
shop/
├── backend/
│   ├── main.go                  # Точка входа, роутер
│   ├── go.mod                   # Go модуль
│   ├── database/
│   │   └── db.go                # Инициализация SQLite, таблицы, seed-данные
│   ├── models/
│   │   └── models.go            # Структуры данных
│   ├── middleware/
│   │   └── auth.go              # JWT аутентификация
│   └── handlers/
│       ├── auth.go              # Регистрация / Вход / Профиль
│       ├── products.go          # Товары и категории
│       ├── cart.go              # Корзина
│       └── orders.go            # Заказы
├── frontend/
│   ├── index.html               # SPA — все 7 страниц
│   ├── css/
│   │   └── style.css            # Дизайн-система
│   └── js/
│       └── app.js               # Вся логика фронтенда
└── start.sh                     # Скрипт запуска
```

## 🚀 Быстрый старт

### Требования
- Go 1.21+
- CGO (для SQLite — обычно включён по умолчанию)

### Установка и запуск

```bash
# 1. Войти в папку проекта
cd shop

# 2. Установить зависимости и запустить бэкенд
chmod +x start.sh
./start.sh
```

Или вручную:

```bash
cd backend
go mod tidy
go run main.go
```

### Открыть в браузере

После запуска перейдите на:
```
http://localhost:8080
```

> Фронтенд отдаётся самим Go-сервером как статические файлы.  
> База данных `shop.db` создаётся автоматически в папке `backend/`.

## 🛠 API Endpoints

| Метод | Путь | Описание | Auth |
|-------|------|----------|------|
| POST | `/api/auth/register` | Регистрация | — |
| POST | `/api/auth/login` | Вход | — |
| GET | `/api/auth/me` | Профиль | ✓ |
| GET | `/api/products` | Список товаров (с фильтрами) | — |
| GET | `/api/products/:id` | Детали товара | — |
| GET | `/api/categories` | Список категорий | — |
| GET | `/api/cart` | Содержимое корзины | ✓ |
| POST | `/api/cart/add` | Добавить в корзину | ✓ |
| PUT | `/api/cart/update` | Изменить количество | ✓ |
| DELETE | `/api/cart/remove` | Удалить из корзины | ✓ |
| POST | `/api/orders` | Создать заказ | ✓ |
| GET | `/api/orders/my` | Мои заказы | ✓ |

### Фильтры для `/api/products`

```
?search=iphone
?category_id=1
?min_price=1000&max_price=50000
?brand=Apple
```

## 🗄 Таблицы БД

| Таблица | Описание |
|---------|----------|
| `users` | Покупатели (id, username, email, password) |
| `categories` | Категории товаров |
| `products` | Товары (цена, остаток, бренд, фото) |
| `cart` | Корзина пользователя |
| `cart_items` | Позиции корзины |
| `orders` | Заказы |
| `order_items` | Позиции заказа |

## 📄 Страницы фронтенда

| Страница | Описание |
|----------|----------|
| **Главная** | Витрина с популярными товарами и баннерами категорий |
| **Каталог** | Все товары, поиск, фильтры по цене/бренду/категории |
| **Карточка товара** | Детали, характеристики, выбор количества, кнопка "В корзину" |
| **Корзина** | Список товаров, изменение количества, удаление, сумма |
| **Оформление заказа** | Адрес доставки, выбор типа доставки, подтверждение |
| **История заказов** | Все заказы со статусами и раскрываемыми деталями |
| **Вход / Регистрация** | JWT-аутентификация |

## 🔐 Аутентификация

Используется **JWT токен** (72 часа). После входа токен хранится в `localStorage` и передаётся в заголовке:

```
Authorization: Bearer <token>
```

## 🎨 Дизайн

- Тёмная тема в стиле **luxury editorial**
- Шрифты: **Cormorant Garamond** (заголовки) + **DM Sans** (текст) + **DM Mono** (метки)
